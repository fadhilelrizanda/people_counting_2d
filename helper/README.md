# Helper Directory

This directory (`helper/`) contains utility scripts to streamline the development workflow.

## Contents
* `download_output.py`: A Python script designed to download the output files (like trained model weights or logs) from a remote Kaggle kernel execution to your local machine.

## Usage
```bash
python helper/download_output.py fadhilelrizandamicr/your-kernel-slug -p ./models
```
